Map Info Section

This section tells the plugin the most basic information about your map. All the following components are required on every map

info:
   name: Exampler
   world: example
   version: '1.0'
   type: tdm
   creators: 'IcyCode, Notch'
   spawn: 
      x: 125
      y: 64
      z: 5
      yaw: 0
      pitch: 0 #Spectator Spawn
